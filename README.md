## v1.4 — Mejoras de usabilidad, OCR y sistema

**Fecha de lanzamiento:** Abril 2026  
**Autor:** danypolo  

### ✨ Novedades

#### 🧠 OCR mejorado y configurable
- Sistema de OCR más estable y preciso  
- Sensibilidad configurable desde la interfaz  
- Mejor detección de números pequeños y HUD variables  
- Adaptación a colores dinámicos (no solo amarillo)  
- Mejor interpretación de valores con separadores (ej: 51,000)  
- Mayor robustez frente a ruido visual  

#### ⚙️ Sistema y rendimiento
- Eliminación automática de logs al iniciar  
- Mejor gestión de errores en OCR  
- Corrección de conflictos entre modos (material / hand)  
- Mejora en la detección de múltiplos de minerales  

#### 🎮 Interfaz y experiencia de usuario
- Hotkey global para calibración (sin botones)  
- Eliminado popup de confirmación al calibrar  
- Mejora del comportamiento tras ALT+TAB  
- Corrección del overlay “always on top”  
- Ajustes visuales (texto ya no se corta)  
- Visualización de la versión actual en pantalla (v1.4)  

#### 🔄 Sistema de actualizaciones
- Comprobación de versión online integrada  
- Lectura automática de `version.json` desde GitHub  
- Notificación cuando hay nueva versión disponible  
- Visualización de cambios dentro de la aplicación  

#### 🛠️ Instalación y distribución
- Mejor integración en el `.exe` (incluyendo hotkeys)  
- Corrección de problemas de permisos  
- Instalador optimizado usando la estructura completa de `dist`  
- Mejor compatibilidad con la librería `keyboard`  

#### 🐛 Correcciones
- Hotkeys que no respondían correctamente  
- Error con `ctypes.wintypes`  
- Problemas al cambiar de ventana (ALT+TAB)  
- Fallos en detección manual de valores  
- Inconsistencias en algunos minerales

## 🇬🇧 English

## v1.4 — Usability, OCR and system improvements

**Release date:** April 2026  
**Author:** danypolo  

### ✨ What's new

#### 🧠 Improved and configurable OCR
- More stable and accurate OCR system  
- Configurable OCR sensitivity from the UI  
- Better detection of small HUD numbers  
- Improved handling of dynamic HUD colors (not only yellow)  
- Better parsing of values like 51,000  
- Increased robustness against visual noise  

#### ⚙️ System and performance
- Automatic log cleanup on startup  
- Improved OCR error handling  
- Fixed conflicts between modes (material / hand)  
- Improved mineral multiple detection  

#### 🎮 UI & user experience
- Global hotkey for calibration (no buttons required)  
- Removed calibration confirmation popup  
- Improved ALT+TAB behavior  
- Fixed always-on-top overlay issues  
- UI adjustments (text clipping fixed)  
- Display of current version (v1.4)  

#### 🔄 Update system
- Integrated online version checking  
- Reads `version.json` from GitHub  
- Notifies when a new version is available  
- Displays changelog inside the app  

#### 🛠️ Installation & distribution
- Improved `.exe` packaging (including hotkeys)  
- Fixed permission issues  
- Optimized installer using full `dist` structure  
- Better compatibility with `keyboard` library  

#### 🐛 Fixes
- Non-working hotkeys  
- `ctypes.wintypes` error  
- ALT+TAB issues  
- Manual input detection bugs  
- Minor mineral detection inconsistencies
